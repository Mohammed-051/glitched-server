A minimalist, dark-toned computer graphics project exploring 2D and 3D geometric transformations with OpenGL and GLUT.

---

## Context

- Senior CSE project
- Daffodil International University
- Focus: interactive transformation concepts in C++

## Stack

- C++17
- OpenGL
- GLUT / FreeGLUT

## Run (example)

```bash
g++ main.cpp -o app -lfreeglut -lopengl32 -lglu32
./app
```

## Roadmap

- 2D translation, rotation, scaling
- 3D transformation pipelines
- Keyboard-driven scene interaction
- Camera and projection experiments
